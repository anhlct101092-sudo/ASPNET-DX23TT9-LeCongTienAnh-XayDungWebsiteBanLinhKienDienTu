# Báo cáo tiến độ tuần 01 - Nền tảng MVC

## Công việc thực hiện

- Phân tích đề tài website bán linh kiện điện tử.
- Xác định tác nhân khách hàng và quản trị viên.
- Lập danh sách chức năng và ràng buộc dữ liệu.
- Thiết kế 16 bảng nghiệp vụ và quan hệ Entity Framework Core.
- Cấu hình ASP.NET Core MVC, SQL Server và ASP.NET Identity.
- Xây dựng Repository, Service và Dependency Injection.
- Xây dựng giao diện trang chủ theo mẫu cửa hàng linh kiện.
- Hiện thực tìm kiếm, lọc, phân trang, giỏ hàng Session và checkout transaction.
- Xây dựng AdminLTE Dashboard, CRUD, upload ảnh và Chart.js.

## Kết quả

Hoàn thành mã nguồn phiên bản MVC .NET 8 với dữ liệu seed, phân quyền và giao diện responsive, đáp ứng nhóm tiêu chí giải pháp và cài đặt trong phiếu đánh giá.

## Kế hoạch tiếp theo

- Cài .NET 8 SDK và SQL Server trên máy phát triển.
- Tạo migration `InitialCreate`, build và chạy toàn bộ test case.
- Hoàn thiện báo cáo 30-50 trang và slide thuyết trình.
